# Sistema de Predição de Métodos Iterativos para Sistemas Lineares

Este repositório contém o código para o TCC de Davi Roriz Oliveira, orientado por Francisco Bruno Souza Oliveira, na Universidade Estadual de Santa Cruz (UESC).

## Instalação
1. Crie um virtualenv: `python -m venv venv`
2. Ative: `source venv/bin/activate` (Linux/Mac) ou `venv\Scripts\activate` (Windows)
3. Instale dependências: `pip install -r requirements.txt`

## Uso
- Rode testes: Rode main.py
- Gere dataset: Rode main.py (Chamando o generate_dataset)
- Treine modelo: Ainda não implementado
